Quickstart¶

Below is a short tutorial that will show you how to get started using MLBlocks.

In this tutorial we will learn how to:

  • Create a pipeline using multiple primitives
  • Specify hyperparameters for each primitive in the pipeline
  • Fit the pipeline using training data
  • Use the pipeline to make predictions from new data

Creating a pipeline¶

With MLBlocks, creating a pipeline is as simple as specifying a list of primitives and passing them to the MLPipeline class:

In [1]: from mlblocks import MLPipeline

In [2]: primitives = [
   ...:     'sklearn.preprocessing.StandardScaler',
   ...:     'sklearn.ensemble.RandomForestClassifier'
   ...: ]
   ...: 

In [3]: pipeline = MLPipeline(primitives)

Optionally, specific hyperparameters can be also set by specifying them in a dictionary:

In [4]: hyperparameters = {
   ...:     'sklearn.ensemble.RandomForestClassifier': {
   ...:         'n_estimators': 100
   ...:     }
   ...: }
   ...: 

In [5]: pipeline = MLPipeline(primitives, hyperparameters)

If you can see which hyperparameters a particular pipeline is using, you can do so by calling its get_hyperparameters method:

In [6]: pipeline.get_hyperparameters()
Out[6]: 
{'sklearn.preprocessing.StandardScaler#1': {'with_mean': True,
  'with_std': True},
 'sklearn.ensemble.RandomForestClassifier#1': {'n_jobs': -1,
  'n_estimators': 100,
  'criterion': 'entropy',
  'max_features': None,
  'max_depth': 10,
  'min_samples_split': 0.1,
  'min_samples_leaf': 0.1,
  'class_weight': None}}

Making predictions¶

Once we have created the pipeline with the desired hyperparameters we can fit it and then use it to make predictions on new data.

To do this, we first call the fit method passing the training data and the corresponding labels.

In [7]: from sklearn.datasets import load_iris

In [8]: from sklearn.model_selection import train_test_split

In [9]: iris = load_iris()

In [10]: X_train, X_test, y_train, y_test = train_test_split(iris.data, iris.target)

In [11]: pipeline.fit(X_train, y_train)

Once we have fitted our model to our data, we can call the predict method passing new data to obtain predictions from the pipeline.

In [12]: from sklearn.metrics import accuracy_score as score

In [13]: y_pred = pipeline.predict(X_test)

In [14]: y_pred
Out[14]: 
array([0, 0, 1, 0, 0, 2, 2, 1, 0, 2, 0, 0, 2, 0, 2, 2, 2, 2, 2, 0, 0, 2,
       2, 0, 1, 0, 0, 0, 0, 2, 1, 1, 0, 1, 2, 2, 1, 1])

In [15]: score(y_test, y_pred)
Out[15]: 0.9473684210526315
